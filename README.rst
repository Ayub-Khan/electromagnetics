How do I run this project?
==========================

**Setup Instructions:**

1. Create a python 3 virtual environment
.. code-block:: bash
    python3 -m venv venv

    source venv/bin/activate

3. Install requirements:
.. code-block:: bash
    make requirements

4. Create database and run migrations:
.. code-block:: bash
    make migrate

5. Serve the dota-world app on localhost at port 4444
.. code-block:: bash
    make serve
